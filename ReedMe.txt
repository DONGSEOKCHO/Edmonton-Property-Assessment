set up steps.
1) open File -> Project Structure
2) Select Modules(from left)
3) Select Dependencies
4) Click Plus button (top right)
5) Click Jar or directory
6) Select json-20201115 jar(if downloaded found it at your downloaded location) and click OK
7) check the json-20201115
8) Click Apply and OK
